# Android приложение для отображения списка товаров
### Функционал:
- Вывод списка товаров с отображением title, description, thumbnail и price
- Пагинация
- Swipe to refresh
- Обработка ошибок

### Используемые технологии и подходы:
- Android View XML
- RxJava
- Retrofit
- Dagger 2
- Glide
- Kotlin Serialization
- Shimmer
- Многомодульность
- MVI
